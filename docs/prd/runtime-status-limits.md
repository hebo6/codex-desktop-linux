# 瞬时状态与限额

## 审批面板

命令、文件修改或其他高风险动作需要审批时，在输入区上方显示固定面板

审批属于瞬时交互状态，不插入主回答历史

侧边栏对应会话和窗口任务栏同时显示待审批提示

面板展示请求类型、命令或文件范围、工作目录、风险说明和服务端上下文

操作至少包括允许一次和拒绝，服务端支持时增加本回合允许或按规则允许

高风险长期授权必须二次确认并明确作用域

审批按到达顺序排队，一次显示一个主审批，并展示剩余数量

提交后按钮进入忙碌状态，其他窗口先完成审批时当前面板转为已在其他窗口处理

关闭窗口或断线不得产生自动批准的界面结果

## 连接与恢复状态

主内容区在连接完成前显示当前阶段，侧边栏服务器固定入口持续显示简短连接状态

顶栏不显示连接状态；连接失败页提供重试和诊断入口，自动重连期间提供立即重试和停止重连

需要用户修改配置的问题停止重试，并提供进入服务器设置的操作

重连开始后保留当前进程已加载内容并显示正在同步状态，服务端响应到达后以服务端事实替换

连接中断时，同一进程已读取的当前服务器和当前会话可以只读查看，并显示最后同步时间

冷启动断线时不显示上次运行的会话正文，进程内保留内容不得写入 SQLite 或其他持久化介质

离线状态下发送、审批、归档和删除操作禁用，并说明原因

## 桌面通知

任务完成、等待审批和连接失败可以发送系统通知

点击通知后聚焦对应窗口和会话

通知内容不得展示凭据或未经用户允许展示的敏感正文

## 错误反馈

| 层级 | 展示方式 |
| --- | --- |
| 字段校验 | 字段下方内联错误 |
| 单项操作失败 | 操作附近提示并保留上下文 |
| 会话级错误 | 主内容顶部内联错误条 |
| 连接级错误 | 连接页面和服务器入口共同提示 |
| 应用不可恢复错误 | 全窗口错误页和重启入口 |

错误文案先说明发生了什么，再说明数据是否受影响，最后提供下一步操作

用户界面不直接显示 JSON-RPC 错误码、Rust 类型名或堆栈

诊断详情可以展示请求类型、服务器、时间、错误码和脱敏技术信息

## 剩余限额

圆环固定显示在主内容顶栏右侧，在会话页和空白页保持可见

圆环高亮弧和中心数值都表示剩余量

存在多个限额窗口时，默认显示最紧迫窗口的剩余百分比

| 剩余状态 | 表现 |
| --- | --- |
| 充足 | 低强调中性色 |
| 较低 | 警告色 |
| 极低 | 错误强调色 |
| 未知 | 虚线中性色 |

常态剩余充足时不使用高饱和绿色

点击圆环显示各窗口名称、剩余百分比、重置时间和账户摘要

详情浮层使用完全不透明背景，各进度条高亮部分同样表示剩余量

重置时间同时显示相对时间和本地绝对时间

加载失败时保留最后成功值、标记更新时间并提供刷新

更新时平滑改变弧长，不使用滚动计数动画

### Token 历史消耗

详情浮层在限额窗口下方显示 Token 历史消耗区域，默认展开并允许用户收起

历史图表最多展示最近 14 个每日用量桶，使用柱状图比较每日 Token 消耗

悬浮柱体时显示对应日期和当日用量摘要，数值达到千、百万或十亿时分别使用 `K`、`M`、`B` 缩写并最多保留一位小数

读取中、读取失败和没有历史数据时分别显示明确状态，不影响剩余限额和重置操作

### 限额重置次数

可用重置次数大于 0 时在详情浮层底部显示可折叠区域，标题展示可用次数且默认收起

服务端提供凭证详情时，逐项显示标题、说明、过期时间或永久有效状态，可用凭证提供使用操作

服务端只提供可用次数而没有凭证详情时显示快速重置操作，明确返回空凭证列表时显示没有可用重置凭证

消耗重置次数前必须确认，重置期间禁用重复操作；重置成功或凭证已经兑换时刷新限额，其他结果和请求失败在详情浮层中说明原因
