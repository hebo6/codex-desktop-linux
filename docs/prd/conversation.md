# 对话

## 顶栏

顶栏固定在主内容区顶部

左侧在窄窗口显示侧边栏按钮，随后显示会话标题和可选项目路径

右侧依次显示连接状态、剩余限额圆环和会话更多菜单

服务端更新会话标题时使用淡入替换，不改变历史消息布局

## 消息流

消息流位于顶栏与输入区之间，内容列水平居中

消息列和输入区使用同一水平内容轨道，在各响应式断点保持左右边缘一致

首条消息上方保留清晰空间，同类型消息间距小于不同角色或活动组间距

用户消息右对齐，AI 回答和活动左对齐

用户消息使用有限宽度气泡，AI 回答不使用气泡或卡片背景，直接在消息列中展示内容

消息操作区在使用精细指针时随悬浮显示，在键盘焦点进入时立即显示，在粗指针环境中直接可见

消息操作按钮默认只显示图标，使用精细指针悬浮按钮或键盘焦点进入按钮时显示文字提示，提示不得改变消息操作区尺寸

消息操作区打开菜单或浮层后保持可见，直到浮层关闭且焦点离开

消息操作区的显隐不得改变消息高度、后续回合位置或当前滚动锚点

只有用户接近底部时才自动跟随新消息，用户离开底部阅读时保持位置并显示回到底部按钮

用户主动发送新回合后，将最新用户消息平滑定位到视口上部的上下文位置，为后续回答保留连续阅读空间

向上加载历史时以首个可见项目为滚动锚点

长会话允许对视口外回合进行虚拟化，未渲染回合必须保留稳定占位，重新进入视口时不得发生滚动跳动

每个回合的持久化项目按服务端顺序展示，时间信息只用于详情和完成摘要，不参与正文排序

可由回合项目恢复的思考和工具运行状态显示在活动组内

连接、重连、网络等待和令牌流速等未形成回合历史的瞬时状态显示在输入区上方的运行状态条，不写入消息历史

## 用户消息

用户消息展示文本、结构化引用和附件摘要

用户消息操作区位于气泡下方并右对齐，依次显示提问时间点和复制按钮

提问时间点与用户消息操作区同步显隐

复制内容使用可读纯文本，并保留文件或技能引用的显示名称

复制成功后图标短暂变为完成状态并向辅助技术播报，随后恢复

图片附件显示缩略图，点击打开图片预览，加载失败时显示文件名和重试

## AI 回答

AI 回答使用安全 Markdown 渲染，流式阶段保持稳定的块级结构，完成后以最终内容校正

流式回答的当前输出末尾显示低强调脉冲点，紧随最后一个可见内容且不单独占行，回合结束后移除，减少动画模式下保持为静态圆点

回答尾部的操作区提供复制和在新会话中继续图标按钮

最新一轮最终回答的复制和在新会话中继续按钮常显，较早最终回答的按钮在使用精细指针悬浮或键盘焦点进入回答时显示

回答时间点不随最新一轮操作按钮常显，在使用精细指针悬浮或键盘焦点进入回答时显示

复制按钮默认复制 Markdown 源文，按住 Shift 点击时复制渲染后的纯文本

在新会话中继续从当前回答处分叉，最新一轮可以立即执行，从较早历史继续前显示确认

分叉成功后在当前窗口打开新会话，并显示来源会话和分叉位置提示

分叉失败时停留在原会话，不创建虚假的本地分支
