# 服务器配置

## 固定入口

侧边栏底部固定显示当前服务器按钮，不随会话列表滚动

按钮包含服务器图标、名称、状态点、简短状态文字和展开箭头

连接状态包括未连接、连接中、初始化中、已连接、重连中和错误

点击按钮在其上方打开服务器选择器，按 Esc 或点击外部关闭

## 服务器选择器

浮层不超过侧边栏和窗口的可用范围，内容超出后内部滚动

顶部显示标题和新建服务器按钮，中部按最近使用顺序展示已保存服务器

每行展示类型、名称、地址或命令摘要、状态，以及连接和在新窗口打开操作

编辑和删除放在行尾更多菜单中

当前服务器行使用选中背景，已连接但不属于当前窗口的服务器只显示连接状态

连接中禁止重复点击，失败时在行内显示简短错误并提供诊断入口

连接成功后关闭浮层、刷新最近会话并恢复该窗口上次打开的会话

当前编辑器有未发送草稿时，切换服务器前提供保留草稿并切换、清空后切换和取消三个选择

## 新窗口连接

在新窗口打开服务器后，新窗口先显示连接骨架屏，再恢复最近会话或显示空白任务页

多个窗口连接同一服务器时，会话选择和草稿相互独立

## 服务器编辑器

服务器编辑器使用模态对话框，第一项选择本机 stdio 或远程 WebSocket

| 类型 | 字段 |
| --- | --- |
| 本机 stdio | 名称、可执行文件路径、参数列表、默认工作目录、普通环境变量、敏感环境变量 |
| 远程 WebSocket | 名称、URL、认证方式、令牌、普通请求头、连接超时、TLS 证书策略、连接路径 |

敏感字段只显示是否已经保存，不回填明文

凭据优先保存到 Linux Secret Service。提交新凭据时若 Secret Service 不可用，保存凭据前显示权限受限明文文件的安全风险，只有用户明确确认后才允许写入；取消确认时保持编辑内容且不写入凭据

保存前提供测试连接按钮，测试不会创建会话

字段错误显示在对应字段下方，WebSocket URL 只接受 `ws` 或 `wss`，本机可执行文件必须是绝对路径

服务器名称不能为空，并且在本机配置中唯一

使用 `ws://` 时显示明文传输警告并要求一次确认

编辑正在使用的服务器并保存后，提示立即重连或稍后应用，并说明受影响窗口

## 代理配置界面

远程服务器必须选择直连或一个已保存代理，代理失败时界面保持失败状态，不静默切换为直连

代理作为独立配置在设置中管理，服务器编辑器可以选择已保存代理或新建代理

代理列表展示类型、地址摘要、引用服务器数量和最近测试结果

代理名称不能为空，并且在本机配置中唯一

| 类型 | 主要字段 |
| --- | --- |
| HTTP CONNECT | 名称、代理 URL、认证方式、用户名、密码或令牌、自定义请求头、连接超时、TLS 策略 |
| SOCKS5 | 名称、主机、端口、认证方式、用户名、密码、DNS 解析方式、连接超时 |
| SSH | 名称、主机、端口、用户名、认证方式、密钥路径、密钥口令、密码、主机密钥、超时和保活 |

密码、令牌和密钥口令默认遮盖，留空表示保持已有值，清除凭据使用独立操作

选择私钥使用系统文件选择器，界面展示完整路径

TLS 证书策略和 SSH 主机密钥策略使用高风险说明，不提供统一关闭全部安全校验的开关

首次连接未知 SSH 主机时显示主机名、端口、密钥算法和 SHA256 指纹

已保存主机密钥发生变化时阻止连接，并同时显示原指纹和新指纹

测试连接分阶段展示进度与耗时

测试详情不得显示密码、令牌、私钥内容或完整认证头

被服务器引用的代理不能删除，删除对话框列出引用服务器并提供前往修改入口

## 删除服务器

删除前显示服务器名称、受影响窗口数量和本地数据清理范围

确认信息明确说明不会删除服务端会话或远程文件

仍被窗口使用的服务器不能删除，用户需要先关闭相关窗口或切换服务器
