# 窗口与导航

## 窗口

窗口标题由当前会话标题和服务器名称组成，没有当前会话时显示产品名称和服务器名称

窗口不使用系统原生标题栏，窗口控制按钮（关闭、最大化、最小化）集成在主内容顶栏中，按钮类型、顺序和位置在启动时从系统 button-layout 配置动态读取

应用记忆窗口尺寸、位置、最大化状态、服务器和会话

窗口位置超出当前显示器范围时回到主显示器可见区域

再次启动应用时激活已有实例，关闭最后一个窗口默认退出应用

## 全局布局

界面由左侧会话栏、中心主内容区和覆盖式浮层组成

宽窗口中侧边栏常驻，分隔线必须易于拖动

窄窗口中侧边栏切换为覆盖式抽屉，关闭抽屉后主内容占满窗口

覆盖式浮层不改变主内容和侧边栏的布局尺寸

## 左侧会话栏

### 区域结构

侧边栏从上到下依次为最近会话区和服务器固定区，不显示顶部品牌标识

侧边栏可滚动区域必须为滚动条预留独立空间，滚动条不得与其左侧的文字、按钮或交互命中区域重合，也不得遮挡内容

设置、帮助、技能和应用不作为常驻导航，设置通过应用菜单和命令入口打开

### 标题栏

标题栏左侧依次显示侧边栏折叠按钮和最近会话，右侧依次显示新建会话和按项目分组按钮

宽窗口通过侧边栏标题中的按钮收起侧边栏，收起后只在主内容顶栏显示恢复侧边栏按钮；窄窗口中的同一按钮关闭覆盖式侧边栏

标题栏固定在侧边栏顶部，列表离开顶部后才显示低强调底部分隔，列表位于顶部时不保留常驻边框

按钮提供 tooltip 和可访问名称

按项目分组激活时使用低强调选中背景

点击标题栏新建会话后主内容切换到空白任务页并沿用当前服务器；当前已打开会话有工作目录时，项目选择器预选该目录，否则使用新建页默认项目，其他参数使用服务器默认值

空白页存在未发送草稿时再次新建，应保留原草稿为窗口级临时草稿，用户返回时可以恢复

### 会话列表

未分组模式按最后更新时间倒序展示

分组模式按项目或工作目录分组，组内按最后更新时间倒序，无工作目录的会话归入其他会话

每个展开的项目组初始显示最近更新的 3 个会话

项目组仍有未显示会话时，在组尾显示加载更多按钮，点击后继续加载最多 3 个会话，各项目组的加载状态相互独立

收起并重新展开项目组时保留本窗口已加载的会话数量

当前会话不在已显示范围内时，自动加载至当前会话可见，不改变会话排序

会话归档或删除后，项目组仍有未显示会话时自动补足当前显示数量

加载失败时保留已有会话，并在加载更多按钮位置提供重试

项目组标题左侧显示折叠箭头和目录名，右侧显示从此项目新建会话按钮；按钮使用低强调样式并始终保留固定命中区域，项目名溢出时显示省略号，悬浮项目名后显示完整路径

点击从此项目新建会话后，主内容切换到空白任务页，沿用当前服务器，并将项目选择器预选为该项目组对应的完整工作目录；发送前仍可更改项目，其他参数使用服务器默认值

从此项目新建会话不改变项目组展开状态，首次发送成功前不向会话列表添加记录

无工作目录的其他会话组不显示从此项目新建会话按钮

窄窗口点击从此项目新建会话后关闭覆盖式侧边栏

按钮提供在此项目新建会话的 tooltip，可访问名称包含完整工作目录

会话行采用紧凑且一致的高度，包含状态图标、单行标题和可选未读或运行状态

标题溢出时显示省略号，短暂悬浮后显示完整标题和项目路径

当前会话使用低强调选中背景和左侧细指示条，不使用高饱和品牌色大面积填充

| 会话状态 | 表现 |
| --- | --- |
| 运行中 | 低强调脉冲点 |
| 等待审批 | 高关注圆点 |
| 失败 | 错误图标 |
| 完成 | 普通样式 |

长列表分页加载不得改变当前滚动锚点

### 会话操作

鼠标进入或键盘焦点进入会话行后，标题尾部显示归档和删除按钮

操作按钮出现时临时缩短标题区域，隐藏时标题延伸到行尾

点击行主体打开会话，中键点击在新窗口打开，双击不产生额外行为

窄窗口点击行主体打开会话后自动关闭覆盖式侧边栏，宽窗口保持侧边栏状态不变

归档成功后以收起动画移出列表，并显示可撤销提示

删除前显示确认对话框，说明会话标题、服务器和不可恢复性

删除进行中禁用该行操作，失败时保持原位置并展示原因

方向键移动焦点，Enter 打开，Delete 触发删除确认

### 空白和错误状态

无会话时显示尚无最近会话和新建任务提示，不重复放置大型新建按钮

列表加载失败时保留当前进程已加载的会话项，在列表顶部显示内联错误和重试按钮

服务器断线时，同一进程内已经加载且仍属于当前服务器和当前会话的内容继续以只读方式显示，禁止执行的操作显示断线原因

应用冷启动或进程退出后不从本地恢复会话列表与会话正文，必须重新连接 app-server
