# 多窗口与设置

## 多窗口

服务器选择器、会话菜单和应用菜单都可以在新窗口打开

新建窗口时使用当前服务器打开空白任务窗口

会话在新窗口打开时，原窗口保持不变

支持的深链交给已有实例处理，并打开对应服务器或会话

深链不得携带或显示明文令牌

每个窗口独立保存当前服务器、会话、草稿、滚动位置和局部 UI 状态

草稿不跨窗口实时同步

归档、删除、标题、连接、限额和审批结果同步到相关窗口

会话在其他窗口被删除后，当前窗口显示已删除状态并提供返回空白页操作

多个窗口同时操作同一会话时不显示虚假的客户端锁定状态，以实际结果反馈为准

## 设置与诊断

设置使用独立窗口或主窗口模态页，采用左侧导航和右侧内容结构

设置分区包括外观、通用、通知、服务器、代理、权限、数据与隐私、快捷键和诊断

设置变更立即生效，或明确标记需要重连或重启

服务器和代理管理复用对应编辑器的界面与校验行为

危险操作集中放在数据与隐私页底部，不与常用开关混排

数据与隐私页分别提供清理日志、临时文件和全部本地数据

清理全部本地数据前显示确认，并明确凭据、草稿和常用提示词也会删除

诊断面板展示客户端版本、协议基线、Linux 环境、WebView 版本、服务器状态、代理类型、连接阶段、最近错误和当前进程最近的会话恢复耗时

会话恢复耗时至少区分响应等待、JSON 解析、协议校验、投影转换和首次界面提交，并展示响应字符数与回合和项目数量

性能诊断只保留有限数量的当前进程记录，不包含服务器地址、会话标识、用户消息或文件正文

支持复制脱敏诊断报告
