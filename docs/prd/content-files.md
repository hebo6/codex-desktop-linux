# 内容与文件

## Markdown

支持段落、标题、强调、删除线、列表、任务列表、引用、表格、分隔线、行内代码、代码块、图片和链接

代码块头部显示语言、复制和折行操作，长代码经过视口时头部保持可见

代码正文使用等宽字体和固定行高，默认横向滚动且不将横向滚动链传递给消息流，用户可以切换折行

表格可以在不改变普通正文宽度的前提下使用更大的可用水平空间，起始边缘仍与正文对齐

表格在可用宽度不足时内部横向滚动，并提供复制全部操作，鼠标悬浮或键盘焦点进入后显示，粗指针环境中直接可见

任务列表只读

原始 HTML 忽略，危险链接或嵌入内容阻止渲染并说明原因

流式阶段未闭合代码块保持代码样式，不反复改变已稳定内容的布局

## 链接行为

| 链接类型 | 行为 |
| --- | --- |
| 当前服务器可读取的文本或源码 | 内置文本预览 |
| 当前服务器可读取的图片 | 内置图片预览 |
| 当前服务器可读取的 Markdown | 渲染与源码双视图 |
| 当前服务器可读取的 PDF | 内置 PDF 预览 |
| 网页 `http` 或 `https` | 确认后使用系统浏览器打开 |
| 未支持文件类型 | 显示元数据并提供另存为 |
| 无法识别或危险协议 | 阻止打开并说明原因 |

网页确认框显示完整域名和规范化 URL，可以在本次运行期间信任该域名

远程路径必须显示对应服务器和工作区，不得呈现为本机文件

## 文件打开

加载期间显示文件名、服务器、路径和骨架内容，失败后提供重试和复制路径

文本超过预览限制时可以只显示开头，并明确标记内容已截断

保存大型文件前，确认框展示服务端报告的大小和保存位置

## 文本查看器

查看器显示文件名、相对路径、语言、编码、换行符、大小和最后修改时间

支持行号、跳转行、查找、复制选区、复制路径、切换折行和另存为

链接携带行号或范围时自动定位并高亮目标行

检测为二进制或编码无效时切换到未知二进制视图

## 图片查看器

支持 PNG、JPEG、GIF、WebP 和经过安全处理的 SVG

提供适应窗口、原始尺寸、放大、缩小、旋转和另存为

缩放以指针位置为中心，只有图片大于视口时才启用拖动平移

## 保存

保存使用系统另存为对话框，默认文件名来自远程路径并清理非法字符

取消或失败后不保留无效目标文件

同名文件使用系统覆盖确认，不静默改名
