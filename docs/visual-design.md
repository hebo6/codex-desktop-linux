# 视觉设计参数

本文集中记录 UI 实现和视觉回归使用的具体尺寸、阈值与时间参数，界面结构和行为要求见 [产品需求文档](prd/README.md)

## 布局尺寸

| 项目 | 参数 |
| --- | --- |
| 默认窗口 | 1440 × 900 |
| 最小窗口 | 960 × 640 |
| 侧边栏宽度 | 默认 288px，可在 248px 到 360px 之间调整 |
| 侧边栏抽屉断点 | 窗口宽度低于 1100px |
| 主内容最小宽度 | 672px |
| 对话内容轨道最大宽度 | 880px，消息列和输入区共用 |
| 主内容顶栏高度 | 56px |
| 输入区高度 | 最小 104px，最大为窗口高度的 40% |
| 建议菜单 | 与输入区同宽，最大高 360px |
| 限额圆环 | 视觉直径 42px，命中区域至少 52px |
| 新回合上下文保留区 | 消息流可用高度的三分之一，且不低于 88px |

## 动画与反馈时间

| 项目 | 参数 |
| --- | --- |
| 常规悬浮和颜色过渡 | 120ms 到 160ms |
| 面板尺寸过渡 | 180ms 到 240ms |
| 会话标题 tooltip 延迟 | 500ms |
| 会话归档收起 | 200ms |
| 复制成功状态 | 1.5 秒 |
| 流式回答脉冲 | 1.25 秒，缩放范围为 1 到 1.25 |
| 停止操作等待提示 | 5 秒 |
| 提交操作反馈 | 100ms 内 |

## 限额颜色阈值

| 剩余量 | 表现 |
| --- | --- |
| 大于 25% | 低强调中性色 |
| 10% 到 25% | 警告色 |
| 小于等于 10% | 错误强调色 |
| 未知 | 虚线中性色 |

## 无障碍验证参数

- 正文和交互控件满足 WCAG 2.2 AA 对比度
- 页面缩放到 200% 时核心功能保持可用，不出现页面级水平滚动
- 系统启用减少透明度时不出现背景模糊、遮罩渐变或半透明叠层
