# 测试计划

## 质量目标

| 指标 | P0 目标 |
| --- | --- |
| 冷启动到基础 UI | 2 秒内 |
| 已保存服务器列表可用 | 1 秒内 |
| 本机连接初始化 | 3 秒内，不含服务端异常 |
| 输入回显延迟 | 16ms 内 |
| 1000 个会话滚动 | 60fps 目标 |
| 含 1000 个历史 ThreadItem 的会话首屏恢复 | 2 秒内可交互 |
| 流式文本更新 | 每秒最多合并渲染 30 次 |
| 常规空闲内存 | 300 MiB 内 |

## 单元测试

- JSON-RPC 路由和请求 ID 生命周期
- Schema 校验和所有 ThreadItem reducer
- 增量文本归并、完成快照校正和去重
- 链接分类、路径规范化和危险协议拒绝
- 限额剩余计算、阈值和多窗口选择
- 草稿键、多窗口隔离和连接引用计数
- 常用提示词校验、排序、版本冲突和全局持久化
- HTTP CONNECT 响应解析、双层 TLS 和代理认证隔离
- SOCKS5 地址编码、认证、远程 DNS 和错误映射
- SSH 认证、主机密钥校验、direct-tcpip 通道和保活状态机
- 代理配置版本、服务器引用约束和禁止回退直连

## 契约测试

使用参考提交生成的完整实验版 app-server Schema 和录制的脱敏消息序列验证初始化、thread、turn、审批、技能、文件搜索和限额

契约测试必须验证 `thread/turns/list` 存在，并验证初始化声明 `experimentalApi` 能力

所有服务端请求都必须得到成功或明确错误响应

协议升级时，契约测试必须能暴露新增必填字段和新增项目类型

## 组件测试

- 会话行悬浮和键盘操作
- 会话恢复只请求最近 10 个完整回合并按游标加载更早历史
- 已收起活动详情不挂载，展开和收起遵循尺寸与内容可见性顺序
- 会话恢复诊断记录各阶段耗时且不包含会话标识或正文
- 服务器选择器
- 代理编辑器
- SSH 主机密钥确认
- `/`、`@`、`$` 输入建议菜单
- 常用提示词选择、草稿保留和增删改查管理
- 审批面板
- 限额详情浮层
- 文件预览
- 展开完成后才显示展开态内容
- 收缩开始前隐藏仅展开态内容

## 端到端测试

使用受控 app-server fixture 和真实本机 stdio app-server 覆盖以下流程

1. 新建服务器并测试连接
2. 新建会话并完成一轮回答
3. 审批命令和文件修改
4. 中断与 steer
5. 断线后保留当前进程只读内容，重连后与服务端状态对账
6. 从历史回答分叉
7. 远程文件预览和另存为
8. 两个窗口共享连接并打开不同会话
9. 归档、删除和跨窗口同步
10. 低限额提示和重置时间
11. 经无认证和需认证 HTTP CONNECT 代理完成连接与重连
12. 经本地 DNS 和代理 DNS 两种 SOCKS5 模式完成连接
13. 经 SSH Agent、私钥和密码认证建立隧道并处理主机密钥变化
14. 代理失败后保持断开且不向目标地址发起直连
15. Secret Service 缺失时仅在用户确认后通过权限受限明文文件保存、重启读取和清除凭据，取消确认不保存，恢复 Secret Service 后继续读取已有明文凭据

## UI 与交互验收

### 服务器与会话

- 用户能保存多个服务器并清晰区分状态
- 用户能连接服务器或在新窗口连接
- 连接测试能够展示阶段和脱敏错误
- 代理连接失败后不会显示已经回退直连的成功状态
- 会话支持分组、分页、归档、删除和键盘操作
- 列表更新不改变当前选择和滚动位置

### 对话与输入

- 当前协议项目都有可用展示或未知活动卡片
- 用户消息可以复制，AI 回答可以复制和分叉
- 已完成历史内容不被瞬时运行状态改写
- 运行中可以明确区分追加和停止
- `/`、`@`、`$` 菜单支持鼠标、键盘和输入法
- 模型、思考程度、工作目录和权限在发送前可确认，默认模型、原编码思考程度和服务器明确配置的默认权限仅用于展示且不进入请求覆盖字段
- 请求结果不确定时不显示重复的本地消息或回合
- 常用提示词可以独立发送或追加，输入框中的草稿、令牌和附件保持不变

### 状态与恢复

- 审批始终显示在主历史之外，并具有明确风险和操作范围
- 断线、重连、离线和需要修改配置的状态可区分
- 断线恢复后消息顺序和活动状态与服务端一致
- 冷启动断线时不恢复上次进程的会话列表或正文
- 多窗口中的服务端事实能够同步，草稿保持独立
- 错误提示说明影响和下一步操作

### 内容与安全

- AI 回答安全渲染 GFM Markdown
- 网页链接确认后使用系统浏览器打开
- 常见文本、源码、Markdown、JSON、图片和 SVG 可以预览
- 未知类型提供安全另存为
- 远程路径不会被表现为本机路径
- 界面和诊断报告不显示令牌、代理凭据或 SSH 认证数据
- Secret Service 不可用且提交新凭据时显示明文风险确认，一般使用过程不显示该提示
- Secret Service 在前端探测后失效时，未确认的写入由 Rust 边界拒绝且不创建明文文件
- 明文凭据目录和文件权限分别为 `0700` 与 `0600`，符号链接、非当前用户所有和权限过宽的路径均被拒绝

### 限额与可访问性

- 圆环、数值和详情进度条全部表达剩余量
- 只在剩余量较低时提升颜色关注等级
- 多限额窗口默认展示最紧迫项并能查看完整详情
- 键盘可以完成连接、新建、发送、审批、切换会话和打开设置
- 页面放大后核心功能仍可用
- 减少动画模式下不出现非必要位移和缩放

## P0 质量门槛

- 用户能保存至少 20 个服务器配置并清晰区分状态
- Linux 目标发行版完成冒烟验证
- 关键流程具备契约测试和端到端测试
- 日志和诊断报告不包含令牌、代理凭据、SSH 认证数据、用户消息正文和文件正文
- 分页、虚拟滚动和断线恢复不会改变可见内容顺序或当前滚动锚点
- 页面缩放到 200% 时核心功能保持可用，不出现页面级水平滚动
