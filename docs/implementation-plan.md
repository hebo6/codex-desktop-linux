# 实施计划

## 阶段一 基础壳与协议

建立 Tauri 多窗口壳、实验版 Schema 生成、JSON-RPC 路由、未知服务端请求错误响应、本机 stdio、远程 WebSocket、HTTP CONNECT、SOCKS5、SSH direct-tcpip、初始化和诊断日志

## 阶段二 服务器与会话

实现服务器与代理配置、凭据、主机密钥记录、服务器选择器、连接共享、thread 列表、恢复、分页、归档、删除和窗口状态

## 阶段三 核心对话

实现 turn 创建、通知归并、全部 ThreadItem、流式回答、活动卡片、滚动锚点、停止和 steer

## 阶段四 输入与配置

实现结构化编辑器、`/`、`@`、`$` 菜单、附件、模型、思考程度、工作目录、权限和草稿

## 阶段五 安全内容能力

实现 Markdown 安全渲染、统一 LinkResolver、远程文件读取、文本与图片预览、另存为和网页确认

## 阶段六 审批、限额与恢复

实现服务端请求交互、审批队列、剩余限额圆环、断线重连、离线只读缓存和多窗口一致性

## 阶段七 发布质量

完成无障碍、性能、视觉回归、契约与端到端测试、Linux 安装包、签名和发布说明
